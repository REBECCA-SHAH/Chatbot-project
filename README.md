Overview

This project is a real-time AI chatbot interface that delivers context-aware responses using the Google Gemini API. The application focuses on clean UI, low latency responses, and reliable API handling.

# Tech Stack

Frontend: HTML5, CSS3, JavaScript

AI Integration: Google Gemini API

Deployment: Vercel, GitHub Pages

# Features

Real-time chat interface with dynamic message rendering

Context-aware AI responses

Responsive design for desktop and mobile devices
How It Works

User inputs a query

Request is sent to the Gemini API

API processes context and generates a response

Response is dynamically rendered on the UI

# Performance

Handles 500+ daily queries

Low latency response handling

Stable performance through proper error and rate-limit management

# Future Enhancements

Conversation history storage

User login system

Multi-language support

Voice-based interaction
